Welcome to synehr's documentation!
==================================
SYNEHR is a python package that generates synthetic identified EHR Data. The current version (0.0.1) generates Name,
Address, DOB and Race of a subject. It also induces the following types of errors:

1. Character Insertion

2. Character Omission

3. Character Substitution

4. Gender Misclassification

5. Character Transposition


Requirements:

Python Version: 2.7

Installation Requires:

Numpy

Pandas

Fake-Factory

DateTime
